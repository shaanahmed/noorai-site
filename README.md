## Deployment Steps (GitHub Pages)

1. Push the site repo to GitHub.
2. Go to repository Settings.
3. Open Pages.
4. Set source to deploy from the main branch.
5. Save.
6. Wait for GitHub Pages to publish.
7. Open the live site and verify all links work.
8. Use the final privacy policy URL in your app release build and Play Console.
